Romualdus Salernitanus Chronicon Synoptic two-layers visualization

← Back to the edition home page

Details

This page includes synoptic two-layers (graphematic and alphabetic) HTML visualizations of the first three paragraphs (g3.2-3.6, g3.7-3.16 and g3.17-4.2) of the Chronicon by Romualdus Salernitanus in manuscripts A, B and C (sigla codicum).

This is a demo. The rest of the text has been transcribed from manuscripts at the alphabetic layer only.

Source code

See the GitHub repository of the edition for:

  • TEI XML source files (xml folder in the repository)
  • Python software modules (python folder)
  • The graphematic tables of signs (files a-tos.csv, b-tos.csv, c-tos.csv in the csv folder)
  • The tables of standard abbreviation combinations (files a-combi.csv, b-combi.csv, c-combi.csv in the csv folder)
